Opis szczegółowy

klasa Menu

Klasa Menu jest odpowiedzialna za tworzenie menu w uporządkowany sposob. Obiekty tej klasy sa wykorzystywane stricte w klasie AppController. W klasie Menu znajduje sie kontener zawierajacy obiekty klasy MenuOption. Klasa ta nie posiada konstruktora, a więc konieczne jest zainjalizowanie obiektu klasy Menu kontenerem obiektów klasy MenuOption poprzez wywołanie funkcji initMenu().

Inicjalizuje obiekt menu.
