Opis szczegółowy

klasa MenuOption

Klasa MenuOption reprezentuje opcje w menu. Znajduje sie tutaj także obiekt funkcyjny std::function, który określa, jaka funkcja ma byc wykonywana przez obiekt MenuOption. Klasa ta jest mocno związana z klasą menu, gdyż obiekt klasy Menu powinien posiadać kontener obiektów klasy MenuOption.
